🚀 БЫСТРЫЙ СТАРТ

1. Настройте БД:
   echo "DATABASE_URL=postgresql://..." > .env

2. Установите зависимости:
   npm install

3. Примените схему:
   npx drizzle-kit push

4. Запустите:
   npm run dev

5. Инициализируйте:
   curl -X POST http://localhost:3000/api/init

6. Откройте:
   http://localhost:3000
